Mi Glosario Personal De Conceptos De Redes y Comunicaciones

Por: Angel David Ayala Narvaez



A continuación encontrarás las definiciones de las principales palabras y expresiones relacionadas con los conceptos de redes y comunicaciones


ALGORITMO HASH

Un algoritmo hash es una función matemática que toma una entrada (o 'mensaje') y devuelve una salida de longitud fija, que generalmente se representa como una secuencia de números o letras. Esta salida se conoce como el "hash" o "resumen" del mensaje original. Los algoritmos hash se utilizan comúnmente en la criptografía y en la seguridad de la información para verificar la integridad de los datos y para generar firmas digitales.



AM (AMPLITUD MODULADA)

La amplitud modulada (AM) es un método de modulación en el que la amplitud de una señal portadora se modifica de acuerdo con la amplitud de la señal de entrada. Se utiliza en la transmisión de radio para llevar señales de audio, como la música y la voz, a través del aire.



ANCHO DE BANDA

El ancho de banda se refiere a la cantidad máxima de datos que pueden ser transferidos a través de una conexión de red en un período de tiempo dado, generalmente medido en bits por segundo (bps) o en sus múltiplos (kilobits por segundo, megabits por segundo, etc.). Es una medida de la capacidad de la conexión para transmitir información.



AP (ACCESS POINT)

Un access point (punto de acceso) es un dispositivo de red que permite a los dispositivos inalámbricos conectarse a una red cableada para acceder a Internet u otros recursos de red.



ARPANET

ARPANET fue la primera red en utilizar protocolos TCP/IP, sentando las bases de Internet.Concebida como un proyecto de investigación, su objetivo era facilitar la comunicación descentralizada entre investigadores y científicos ubicados en diferentes regiones. Esta red pionera desempeñó un papel fundamental en el intercambio de información a nivel global.



BACKBONE

El backbone en redes es la columna vertebral de la infraestructura de red, proporcionando conexiones de alta velocidad entre diferentes partes de la red para facilitar la transferencia eficiente de datos.



CABLE COAXIAL

El cable coaxial es un tipo de cable que se usa para llevar señales de televisión y de Internet a nuestros hogares. Es como el cable que conecta tu antena de TV o módem a la pared.



CAMPOS ELECTROMAGNETICOS

Los campos electromagnéticos son áreas de energía creadas por la interacción de electricidad y magnetismo. Estas áreas son invisibles y se extienden alrededor de dispositivos eléctricos y cables, así como también a través de ondas de radio y microondas.



DARPA

Defense Advanced Research Projects Agency, fue creada en 1958, la mision principal de Darpa es llevar a cabo investigaciones y desarrollo de tecnologias avanzadas con aplicaciones militares en campos como:IA, robotica, cibernetica, biotecnologia e informatica



DIRECCION DINAMICA

Una dirección dinámica en redes es una dirección IP asignada automáticamente a un dispositivo por un servidor DHCP, lo que facilita la gestión y la conectividad en redes grandes.



DIRECCION ESTATICA

Una dirección estática en redes es una dirección IP que se asigna manualmente a un dispositivo y permanece constante. No cambia automáticamente como lo haría una dirección dinámica asignada por un servidor DHCP. Las direcciones estáticas son útiles para dispositivos que necesitan mantener una identificación constante en la red, como servidores o dispositivos de red críticos.



DIRECCION MAC

La dirección MAC es un identificador único asignado a cada dispositivo conectado a una red de computadoras. Es como el número de serie de un dispositivo en particular y se utiliza para identificarlo de manera exclusiva en la red.



DNS

DNS, o Sistema de Nombres de Dominio, es como una guía telefónica de Internet. Traduce los nombres de dominio que usamos, como "google.com", en direcciones IP que las computadoras pueden entender para encontrar y acceder a los sitios web y otros servicios en línea.



DOMINIO INTERNET

Un dominio en Internet es la dirección única de una página web, equivalente a la ubicación de una casa en una ciudad. Al escribirlo en la barra de direcciones del navegador, accedes a un sitio específico. Los dominios, como "www.ejemplo.com", actúan como identificadores únicos para facilitar la localización y visita de sitios web en la extensa red de Internet.



ENLACE SERIAL

Un enlace serial es una conexión de comunicación que transmite datos de manera secuencial, un bit tras otro, a través de un único canal. Se utiliza comúnmente en redes para conectar dispositivos distantes entre sí, como routers, switches o módems, a través de líneas de comunicación dedicadas, como cables seriales o conexiones de fibra óptica.



ETHERNET

Ethernet es un tipo de tecnología de red que permite la comunicación entre dispositivos, como computadoras, dentro de una red local. Utiliza cables físicos para transmitir datos a alta velocidad.



FIBRA OPTICA

La fibra óptica es un tipo de cable de transmisión que utiliza pulsos de luz para enviar datos a través de largas distancias a alta velocidad.



FIREWALL

Un firewall es un sistema de seguridad que protege una red al filtrar y bloquear el tráfico no autorizado desde y hacia Internet.



FM (FRECUENCIA MODULADA)

La frecuencia modulada (FM) es un método de modulación en el que la frecuencia de una señal portadora se modifica de acuerdo con la amplitud de la señal de entrada. Se utiliza en la transmisión de radio para llevar señales de audio, como la música y la voz, a través del aire.



GAMA RGB

La gama RGB es un sistema de colores que utiliza rojo, verde y azul para representar una amplia variedad de colores. Cada color se define por la cantidad de estos tres colores primarios que contiene. Se utiliza en pantallas y dispositivos digitales para mostrar imágenes y videos.



HERTZ

la unidad de medida de frecuencia en el Sistema Internacional de Unidades y mide el número de veces que se repite un evento durante un segundo



HOSTING

El hosting se refiere al servicio que permite a individuos y organizaciones almacenar y hacer accesibles sus sitios web y archivos en servidores conectados a Internet. Estos servidores, proporcionados por empresas de hosting, alojan el contenido web, permitiendo que sea visible en línea para los usuarios de Internet.



HTTP

HTTP es el protocolo utilizado para la transferencia de datos en la web. Permite que los navegadores soliciten y reciban información de los servidores web.



IMP

La Interface Message Processor (IMP) fue clave en la implementación inicial de ARPANET, la precursora de Internet. Estos dispositivos, desarrollados por BBN, conectaban y facilitaban la comunicación entre computadoras. Cada nodo en ARPANET tenía un IMP que gestionaba el enrutamiento de mensajes. El primer IMP se instaló en UCLA en 1969, marcando un hito fundamental en el desarrollo de las redes de computadoras.



INTERNET RELAY CHAT

IRC es un protocolo de chat en tiempo real que permite la comunicación entre usuarios a través de canales de chat en línea.



INTERNETWORK

Un internetwork es la interconexión de múltiples redes individuales, permitiendo la comunicación entre dispositivos en diferentes ubicaciones.



IP

La dirección IP (Internet Protocol) es un número único asignado a cada dispositivo conectado a Internet, funcionando como su identificación. Este número facilita la ubicación del dispositivo en la red y permite la eficiente transmisión de información entre dispositivos conectados. En resumen, la dirección IP actúa como una "etiqueta" única para guiar el flujo de datos en la red.



LAN

LAN (Red de Área Local) es una red informática que conecta dispositivos en una área geográfica limitada, como una oficina, un edificio o un campus universitario.



MAN

MAN (Red de Área Metropolitana) es una red de comunicación que cubre un área geográfica más grande que una LAN pero más pequeña que una WAN, como una ciudad o una región metropolitana.



MATRIZ TRIDIMENSIONAL

Una matriz tridimensional es un conjunto de datos organizados en tres dimensiones, similar a una cuadrícula o una tabla con profundidad. Cada elemento de la matriz tiene una ubicación única identificada por tres índices: fila, columna y profundidad. Se utiliza en campos como la informática gráfica, la física y la ingeniería para representar información espacialmente.



MECANISMOS DE CHECKSUM

Los mecanismos checksum son métodos utilizados para verificar la integridad de los datos transmitidos. Consisten en calcular un valor de suma o hash a partir de los datos y adjuntarlo al mensaje. Al recibir el mensaje, el destinatario realiza el mismo cálculo y compara el resultado con el valor checksum adjunto para detectar cualquier error de transmisión.



MEDIOS DE TRANSMISION

Los medios de transmisión se refieren a los canales físicos o inalámbricos utilizados para enviar datos entre dispositivos o sistemas de comunicación. Estos medios pueden incluir cables de cobre, fibras ópticas, ondas de radio u otros medios que permiten la transferencia de información desde una fuente a un destino.



MODELO DE REFERENCIA ISO

describe cuáles son los procesos que se han de llevar a cabo para que la comunicación funcione a través de una red.



NCP

Network Control Program, fue un protocolo de comunicacion utilizado en los primeros dias de Arpanet, antes de la adopcion generalizada de TCP/IP, NCP facilito la transferencia de datos entre computadores en la red, con el tiempo, Arpanet y otras redes migraron a TCP/IP



NUBE

En redes, "la nube" se refiere a recursos informáticos y de almacenamiento disponibles a través de Internet, como servicios en línea y almacenamiento remoto.



ONDAS MICROONDAS

Las microondas son un tipo de onda electromagnética que tiene una longitud de onda más corta que las ondas de radio pero más larga que la luz infrarroja. Se utilizan en una variedad de aplicaciones, como la comunicación inalámbrica, la cocción de alimentos en hornos de microondas y la radioterapia en medicina.



PDA

Un PDA es un dispositivo electrónico portátil utilizado para tareas de organización personal, como agenda, calendario y administración de contactos.



PROTOCOLOS FTP

FTP, o Protocolo de Transferencia de Archivos, actúa como el servicio postal de Internet, permitiendo la carga y descarga de archivos entre dispositivos a través de un servidor en la red. Es como un mensajero que facilita el envío eficiente de carpetas y documentos entre computadoras, posibilitando el intercambio y respaldo de archivos entre dispositivos conectados a Internet.



PROVEEDORES DE SERVICIO DE INTERNET(PSI)

Los proveedores de servicios de Internet (ISP) son empresas que ofrecen acceso y conectividad a Internet a usuarios y organizaciones. Estos proveedores permiten a los clientes conectarse a la red global, facilitando el acceso a servicios en línea, navegación web y otras actividades digitales.



QoS

QoS (Quality of Service) en redes se refiere a las tecnologías que garantizan un rendimiento óptimo y una calidad adecuada para la transmisión de datos, priorizando el tráfico y gestionando la congestión de la red.



RED CYCLADES

La red de ordenadores CYCLADES fue una red de investigación francesa de los primeros años de los 70. Pionera en experimentar con la conmutación de paquetes, se creó para explorar alternativas al diseño ARPANET.



RED JERÁRQUICA

Una red jerárquica es una estructura organizativa en capas, donde el nivel superior gestiona el tráfico principal. Capas intermedias facilitan la dirección y organización de información, mientras que en la parte inferior están los dispositivos finales conectados directamente a la red. Este enfoque jerárquico optimiza la eficiencia y gestión de las redes.



RED MALLA

Una red en malla es una estructura donde cada nodo se conecta directamente con todos los demás nodos, permitiendo múltiples rutas para la transmisión de información. Esta configuración mejora la robustez y resistencia a fallos, siendo común en entornos donde la fiabilidad es crucial.



ROUTER

Un router es un dispositivo que conecta redes y dirige el tráfico de datos entre ellas, ayudando a determinar la mejor ruta para la transmisión de datos.



ROUTER INALAMBRICO

Un router inalámbrico es un dispositivo que conecta redes y dirige el tráfico de datos entre ellas, permitiendo conexiones inalámbricas a dispositivos como computadoras, teléfonos inteligentes y tabletas a través de Wi-Fi.



SMTP(CORREO ELECTRÓNICO)

SMTP, o Protocolo Simple de Transferencia de Correo, es como el cartero de Internet para el correo electrónico. Facilita el envío de mensajes desde tu dispositivo a un servidor de correo electrónico, asegurando la entrega eficiente de correos electrónicos a la dirección correcta en la red. Es esencial para una comunicación rápida y efectiva a través del correo electrónico.



SUBDOMINIO

Un subdominio de Internet es una extensión de un dominio principal y se utiliza para organizar o dividir áreas específicas de un sitio web.



SWITCH

Un switch es un dispositivo de red que conecta múltiples dispositivos en una red local y dirige el tráfico de datos de manera eficiente.



TCP

Transmission Control Protocol, es un protocolo de comunicacion de la capa de transporte que proporciona una transmision confiable y orientada a la conexion en redes de computadoras su funcion principal es asegurar la enstrega ordenada y sin errores de datos entre dispositivos concectados a traves de la red, TCP forma parte del conjunto de protocolos TCP/IP que es fundamental para el funcionamiento de internet



TCP/IP

TCP/IP, o Protocolo de Control de Transmisión/Protocolo de Internet, es un conjunto de reglas que permite la comunicación entre dispositivos en Internet. Actúa como el idioma de las computadoras, facilitando la comunicación y el intercambio de información a nivel mundial, asegurando que los mensajes lleguen a sus destinos adecuados, similar a un sistema de correos electrónico en la Internet.



URL

Una URL (Uniform Resource Locator) es una dirección web que identifica la ubicación específica de un recurso en Internet, como una página web o un archivo, y proporciona un medio para acceder a él.



UTP

El cable UTP (par trenzado sin blindaje) es un tipo de cable de red que se utiliza comúnmente en las redes de computadoras. Está compuesto por pares de cables de cobre trenzados para reducir la interferencia eléctrica y es ampliamente utilizado para la transmisión de datos en redes Ethernet.



WAN

Una WAN (Wide Area Network) es una red de computadoras que cubre un área geográfica extensa, como una ciudad, un país o incluso a nivel mundial. Se utiliza para conectar redes locales (LAN) entre sí y permite la comunicación y el intercambio de datos entre dispositivos ubicados en diferentes ubicaciones geográficas.



WLAN

Una WLAN (Wireless Local Area Network) es una red de área local que utiliza tecnología inalámbrica, como Wi-Fi, para conectar dispositivos dentro de un área geográfica limitada, como una casa, una oficina o un campus.



XMPP

XMPP (Extensible Messaging and Presence Protocol) es un protocolo de comunicación en tiempo real utilizado para mensajería instantánea, presencia en línea y otras aplicaciones de colaboración en internet. Es un protocolo abierto y extensible que permite la comunicación entre diferentes servicios y clientes de mensajería compatibles con XMPP.
